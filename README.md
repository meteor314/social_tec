# social_tec

Une application qui permet de partager, des images, partager des moments importants avec ses amis. 
Pour installer l'appliation, vous avez besoin de l'émulateur xampp. 
Puis il faut créer la base de donnée nommée social_tec & importer le fichier SQL dans la base de donnéees.
Où  alors démarrer la machine virtuelbox que je vous ai envoyée. 

Une fois que c'est fait vous pouvez démarrer la VM les ideantifiants sont  :kali, kali. Une fois que c'est fait, ouvrez le terminal puis cltr+alt+t pour ouvrir le Terminal, 
Pour ouvrir VSCODE en admin : 
sudo code --user-data-dir
le mot de passe  : kali

Pour démarrer le serveur : 
sudo /opt/lampp/lampp start
Pour accéder sur au site : 
127.0.0.1/social_tec/
Pour montrer la page de commentaire : 
127.0.0.1/comment.php?id=142
